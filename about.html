GitHub Repository README (Technical Details)

Data Model (ERD)

API Endpoints Used (Read-Only)

  • World Bank GDP: https://api.worldbank.org/v2/country/USA/indicator/NY.GDP.MKTP.CD?format=json
  • World Bank Population: https://api.worldbank.org/v2/country/USA/indicator/SP.POP.TOTL?format=json

Fetched dynamically for external insights; no data persistence.

CRUD Implementation Overview

  • Create: create.html → adds new records to local storage.
  • Read: read.html → searchable / paginated table view.
  • Update: update.html → choose or open via ?id=XX.
  • Delete: delete.html → confirmation-based removal with deep-link support.

MVC Architecture

  • Model: assets/model.js — persistent localStorage service (seed with 50 states).
  • Controller: assets/controller.js — CRUD orchestration & binding.
  • View: HTML pages using modular scripts and assets/charts.js visualizations.

Notable Technical Challenges & Solutions

  • KPI overflow: added auto-scaling typography & ellipsis.
  • Table alignment: fixed layout with table-layout: fixed and ellipsis cells.
  • External insights: switched to World Bank (stable) with latest-year selection.
  • Delete UX: implemented deep-link and preview panel for safer actions.
