About this Project

A lightweight MVC CRUD dashboard for exploring US states. Data persists in localStorage; World Bank API is used for read-only, up-to-date national indicators.

API Endpoints Used (Read-only)

  • World Bank — GDP (current US$): https://api.worldbank.org/v2/country/USA/indicator/NY.GDP.MKTP.CD?format=json&per_page=8&date=2019:2025
  • World Bank — Population: https://api.worldbank.org/v2/country/USA/indicator/SP.POP.TOTL?format=json&per_page=8&date=2019:2025

Logical design used for CRUD entities and relationships.

CRUD Implementation Overview

  • Create: create.html → api.upsert() (localStorage).
  • Read: data.html (search + region filter), index.html aggregates metrics.
  • Update: update.html loads by ID, saves via api.upsert().
  • Delete: delete.html removes via api.remove().

Notable Challenges & Solutions

  • Centering / Consistency: Introduced container-capped inner panels (max-width + margin-inline:auto).
  • External Insights robustness: Fetch recent range with graceful fallback when API returns null.
  • Single source of truth: All UI reads/writes through a singleton api service layered on localStorage.
  • Charts separation: Moved visualizations to charts.html to keep Home fast and focused.
