About this Project

A lightweight MVC CRUD dashboard for exploring US states. Data is stored in your browser (localStorage) and powers both the table and charts. External insights are fetched via the World Bank API (read-only).

API Endpoints Used (Read-only)

  • World Bank — GDP (current US$): https://api.worldbank.org/v2/country/USA/indicator/NY.GDP.MKTP.CD?format=json&per_page=8&date=2019:2025
  • World Bank — Population: https://api.worldbank.org/v2/country/USA/indicator/SP.POP.TOTL?format=json&per_page=8&date=2019:2025

Final logical data model for the application’s entities and relationships.

CRUD Implementation Overview

  • Create: create.html posts to api.upsert() (localStorage).
  • Read: data.html renders from api.list() with search & region filter; index.html aggregates totals.
  • Update: update.html loads record by ID and saves via api.upsert().
  • Delete: delete.html removes a record via api.remove().

Notable Challenges & Solutions

  • Consistent Layout: Content felt left-aligned. Solution: enforce centered containers and panels with margin-inline:auto and fixed max-width.
  • External Insights Reliability: Occasional null/blocked values. Solution: robust World Bank fetch with recent date range and graceful fallbacks.
  • Single Source of Truth: Ensured all pages read/write via a singleton api service on top of localStorage.
  • Charts Separation: Removed Home charts to keep the landing page focused; visualizations live in charts.html.
