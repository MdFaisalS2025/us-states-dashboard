About this Project

A lightweight CRUD dashboard for exploring US states. Data is stored in your browser (localStorage) and powers both the table and charts.

Tech Stack

Vanilla HTML/CSS/JS • ES Modules • Chart.js 4.x

Features

CRUD, Search/Filter, Auto-seeded data, Responsive UI
