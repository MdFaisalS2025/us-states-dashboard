MVC Architecture
  • Model: assets/model.js (localStorage + schema versioning & migration)

  • View: HTML pages + assets/charts.js
Challenges & Solutions
  • KPI blanks → set KPIs separate from chart rendering.
  • Charts not loading → migrated old storage keys to the new one.
  • External API blocked → added World Bank with graceful local fallback.
  • Update UX → added in-page chooser when ?id is missing.
