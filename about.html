About this Project

A lightweight MVC CRUD dashboard for exploring US states. Data persists in localStorage. Read-only national indicators are fetched from the World Bank API.

API Endpoints Used (Read-only)

  • World Bank — GDP (current US$): https://api.worldbank.org/v2/country/USA/indicator/NY.GDP.MKTP.CD?format=json&per_page=8&date=2019:2025
  • World Bank — Population: https://api.worldbank.org/v2/country/USA/indicator/SP.POP.TOTL?format=json&per_page=8&date=2019:2025

Data Model (Updated ERD)

Overview of CRUD Implementation

  • Create: create.html posts via a form into a singleton data service (api.upsert() → localStorage).
  • Read: data.html renders the table with search & region filter; index.html shows aggregates.
  • Update: update.html loads by ID and persists via api.upsert().
  • Delete: delete.html removes via api.remove() and navigates back.

Notable Technical Challenges & Solutions

  • Consistent Centering Across Pages: Introduced .panel-inner with max-width and global centering overrides to prevent left-skew from flex parents.
  • External Insights Reliability: Fetch recent year ranges from World Bank and gracefully handle nulls, ensuring UI never blocks.
  • Single Source of Truth: All UI flows through a singleton api service on top of localStorage for predictable CRUD.
  • Charts Separation: Moved all visualizations to charts.html to keep Home simple and fast.
