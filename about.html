GitHub Repository README (Technical Details)

Data Model (ERD)

API Endpoints Used (Read-Only)

  • World Bank GDP: https://api.worldbank.org/v2/country/USA/indicator/NY.GDP.MKTP.CD?format=json
  • World Bank Population: https://api.worldbank.org/v2/country/USA/indicator/SP.POP.TOTL?format=json
Fetched dynamically for external insights; no data persistence.

CRUD Implementation Overview

  • Create: create.html → adds new records to local storage (Model.upsert).
  • Read: read.html → searchable table (basic), actions for Edit/Delete.
  • Update: update.html → picker + form to modify records.
  • Delete: delete.html → confirmation removal.

MVC Architecture

  • Model: assets/model.js — singleton with schema versioning & persistence.
  • Controller: assets/controller.js — page routers + actions + insights.
  • View: HTML pages with modular scripts and Chart.js visualizations.

Notable Technical Challenges & Solutions

  • KPI blanks: Compute from model aggregates independent of charts.
  • Update UX: Added dynamic chooser + query ?id=XX support.
  • External API reliability: Switched to World Bank with latest non-null fallback and clear errors.
  • Persistence: LocalStorage with schema versioning to avoid stale structures.
